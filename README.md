# Arcanoid Game
---
## RUN
### Depends
- Pyton3.8 or higher
- GUI
- KeyBoard
- Mouse/Touchpad
---
#### Linux / MacOS
```commandline
python3.12 -m venv venv
source venv/bin/activate
pip install -r requriments.txt
```
#### Windows
```commandline
python -m venv venv
venv\Scripts\activate.bat
pip install -r requriments.txt
```
